# 评测服务接口说明

本文档定义评测服务的 API 接口，包括评测集管理和评测任务管理。

---

## 快速参考

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 上传评测集 | POST | `/open/api/v1/evalset` | 上传评测数据 |
| 下载评测集 | GET | `/open/api/v1/evalset/:id` | 分批下载评测数据 |
| 查询评测集数量 | GET | `/open/api/v1/evalset/:id/count` | 查询条目总数 |
| 获取模型列表 | GET | `/open/api/v1/models` | 获取可用推理模型列表 |
| 提交评测任务 | POST | `/open/api/v1/eval/tasks` | 创建评测任务 |
| 查询任务列表 | GET | `/open/api/v1/eval/tasks` | 分页查询任务 |
| 查询任务详情 | GET | `/open/api/v1/eval/tasks/:id` | 获取任务状态和产物 |
| 取消任务 | DELETE | `/open/api/v1/eval/tasks/:id` | 取消运行中任务 |
| 查询任务产物 | GET | `/open/api/v1/eval/tasks/:id/artifacts` | 获取产物列表 |
| 查询评测报告 | GET | `/open/api/v1/eval/tasks/:id/report` | 获取报告摘要 |

---

## 1. 概述

### 1.1 统一响应格式

所有接口返回 JSON 格式：

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `code` | `int` | 业务状态码。`0` 表示成功，非 `0` 表示失败 |
| `message` | `string` | 状态描述信息 |
| `data` | `object` | 业务数据（仅成功时返回） |

### 1.2 分页响应格式

分页列表接口增加 `total` 字段：

```json
{
  "code": 0,
  "message": "success",
  "data": [],
  "total": 100
}
```

**分页请求参数**：

| 参数 | 类型 | 必填 | 默认值 | 约束 | 说明 |
|------|------|------|--------|------|------|
| `page` | `int` | 否 | `1` | `≥ 1` | 页码 |
| `page_size` | `int` | 否 | `20` | `1 ~ 100` | 每页数量 |

### 1.3 认证方式

| Header | 必填 | 说明 |
|--------|------|------|
| `Authorization` | 是 | `Bearer <access_token>` |
| `Content-Type` | 是 | `application/json` |

**认证失效处理**：

```json
{
  "code": 10002,
  "message": "鉴权Token已失效，请重新登录授权"
}
```

### 1.4 错误码

#### 通用错误码 (10000-19999)

| 错误码 | HTTP状态码 | 说明 |
|--------|------------|------|
| `10000` | 500 | 内部服务错误 |
| `10001` | 400 | 请求参数无效 |
| `10002` | 401 | 未授权 |
| `10003` | 403 | 权限不足 |
| `10004` | 404 | 资源不存在 |

#### 评测集错误码 (80000-89999)

| 错误码 | HTTP状态码 | 说明 |
|--------|------------|------|
| `80001` | 400 | 评测集参数无效 |
| `80002` | 500 | 评测集上传失败 |
| `80003` | 500 | 评测集下载失败 |
| `80004` | 500 | 评测集数量查询失败 |

#### 评测任务错误码 (90000-99999)

| 错误码 | HTTP状态码 | 说明 |
|--------|------------|------|
| `90001` | 400 | 任务定义无效 |
| `90002` | 500 | 任务提交失败 |
| `90003` | 404 | 任务不存在 |
| `90004` | 500 | 任务停止失败 |
| `90005` | 404 | 任务产物不存在 |
| `90007` | 409 | 已有运行中任务，需稍后提交 |

## 2. 评测集接口

评测集（Evalset）是评测任务的数据源，包含需要评测的对话条目。

### 2.1 上传评测集

**接口URL**：`POST /open/api/v1/evalset`

**请求参数**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `evalset_id` | `string` | 否 | 评测集唯一标识（不传则自动生成） |
| `items` | `array[EvalsetItem]` | 是 | 评测数据条目列表（1-500条） |

**EvalsetItem 定义**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `case_id` | `string` | 是 | 用例唯一标识 |
| `model` | `string` | 是 | 模型标识 |
| `question` | `string` | 是 | 用户问题 |
| `answer` | `string` | 否 | 模型回答（为空时由推理服务生成） |
| `system` | `string` | 否 | 系统提示词 |
| `history` | `string` | 否 | 历史对话（JSON字符串） |
| `context` | `string` | 否 | 上下文信息 |
| `category` | `string` | 否 | 分类标签 |
| `reference` | `string` | 否 | 参考答案 |
| `keypoint` | `string` | 否 | 评测点列表（JSON字符串数组） |
| `extends` | `object` | 否 | 自定义扩展字段 |
| `metainfo` | `object` | 否 | 元信息 |

**请求示例**：

```json
{
  "evalset_id": "my-evalset-001",
  "items": [
    {
      "case_id": "case-001",
      "model": "gpt-4",
      "question": "什么是大语言模型？",
      "answer": "大语言模型是...",
      "reference": "参考答案"
    }
  ]
}
```

**响应参数**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `data.received` | `int` | 实际接收的条目数量 |

---

### 2.2 下载评测集

**接口URL**：`GET /open/api/v1/evalset/:id`

**路径参数**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | `string` | 是 | 评测集标识 |

**查询参数**：

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `offset` | `int` | 否 | `0` | 起始偏移量 |
| `limit` | `int` | 否 | `300` | 返回条目数（1-500） |

**请求示例**：`GET /open/api/v1/evalset/my-evalset-001?offset=0&limit=100`

---

### 2.3 查询评测集数量

**接口URL**：`GET /open/api/v1/evalset/:id/count`

**响应参数**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `data.count` | `int` | 评测集条目总数 |

### 2.4 获取模型列表

**接口URL**：`GET /open/api/v1/models`

**响应参数**：

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "models": ["deepseek-r1", "gpt-4", "claude-3"]
  }
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `data.models` | `array[string]` | 可用推理模型名称列表 |

## 3. 评测任务接口

评测任务（Eval Task）是评测的执行单元。

### 3.1 提交评测任务

**接口URL**：`POST /open/api/v1/eval/tasks`

**请求参数**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `apiVersion` | `string` | 是 | 工作流模版版本（如 `v1`） |
| `models` | `array[TaskModel]` | 否 | 参与评测的模型列表 |
| `agents` | `array[TaskAgent]` | 否 | 评委列表 |
| `spec` | `object(TaskSpec)` | 是 | 任务执行步骤定义 |

**TaskModel 定义**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | `string` | 否 | 模型唯一标识 |
| `name` | `string` | 否 | 模型显示名称 |
| `type` | `string` | 否 | 模型类型（如 `openai`） |
| `api_key` | `string` | 否 | API密钥 |
| `api_url` | `string` | 否 | API地址 |
| `model` | `string` | 否 | 模型名称 |
| `concurrency` | `int` | 否 | 并发调用数 |
| `params` | `object` | 否 | 模型推理参数 |

**TaskAgent 定义**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | `string` | 否 | 评委唯一标识 |
| `name` | `string` | 否 | 评委显示名称 |
| `type` | `string` | 否 | 评委类型 |
| `api_key` | `string` | 否 | API密钥 |
| `api_url` | `string` | 否 | API地址 |
| `flow_id` | `string` | 否 | 关联流程ID |
| `concurrency` | `int` | 否 | 并发调用数 |
| `params` | `object` | 否 | 评委参数 |

**TaskSpec 定义**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `templates` | `array[TaskTemplate]` | 是 | 步骤模版列表（至少1个`evaluation`类型） |

**TaskTemplate 定义**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `name` | `string` | 是 | 步骤显示名称 |
| `type` | `string` | 是 | 步骤类型 |
| `parameters` | `object` | 否 | 步骤参数 |

**TaskTemplate.type 枚举**：

| 类型 | 必填 | 说明 |
|------|------|------|
| `evalset` | 否 | 评测集准备步骤 |
| `inference` | 否 | **模型推理步骤（评测集无答案时使用）** |
| `evaluation` | 是 | 模型评测步骤（必需） |
| `report-upload` | 否 | 报告上传步骤 |
| `database` | 否 | 数据入库步骤 |

**inference 类型 parameters 定义**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `evalset` | `string` | 是 | 评测集标识（填充实际 evalset_id） |
| `models` | `array[InferenceModel]` | 是 | 推理模型列表 |

**InferenceModel 定义**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | `string` | 是 | 模型服务标识 |

**配置示例**：

```json
{
    "name": "模型推理",
    "type": "inference",
    "parameters": {
        "evalset": "eval-bw7adghvb",
        "models": [
            {"id": "2402509314320990"},
            {"id": "2402509314320991"}
        ]
    }
}
```

**EvalItem 定义**（evaluation步骤的parameters.eval字段）：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `name` | `string` | 否 | 评测项名称 |
| `type` | `string` | 否 | 评测类型（`llm-score`/`agent-score`/`builtin`） |
| `judge_id` | `string` | 否 | 评委ID |
| `params` | `object` | 否 | 评测参数 |
| `prompt` | `object` | 否 | 提示词配置 |
| `weight` | `float64` | 否 | 权重 |
| `func` | `string` | 否 | 评测函数（builtin类型时必填） |

**Prompt 定义**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `role` | `string` | 角色设定 |
| `definition` | `string` | 维度定义 |
| `instruct` | `string` | 评分标准 |
| `step` | `string` | 评测步骤 |
| `body` | `object` | 用例级评测字段注入配置，如 `{"keypoint": true, "context": true}` |

**请求示例**：

```json
{
  "apiVersion": "v1",
  "models": [
    {
      "id": "ID_AJ001",
      "name": "DeepSeek-R1",
      "type": "api-openai",
      "model": "xdeepseekr1",
      "concurrency": 50,
      "params": {
        "max_tokens": 16384,
        "temperature": 0
      }
    }
  ],
  "spec": {
    "templates": [
      {
        "name": "模型评测",
        "type": "evaluation",
        "parameters": {
          "evalset": "eval-bw7adghvb",
          "eval": [
            {
              "name": "相关性",
              "type": "llm-score",
              "judge_id": "ID_AJ001",
              "weight": 0.3,
              "prompt": {
                "definition": "助手响应是否和用户问题强相关",
                "instruct": "5分：内容紧扣核心主题...",
                "step": "请遵循下面的步骤进行评判..."
              }
            }
          ]
        }
      }
    ]
  }
}
```

## 4. 模型接口

模型接口用于查询开放 API 当前可用的文本对话模型列表。模型信息来自主业务数据库中的模型存储表和模型扩展表。

### 4.1 查询模型列表

分页查询模型列表，支持按名称/描述搜索、按文本对话模型类型过滤，以及按是否支持推理过滤。

**请求**

```
GET /models
```

**请求头**

无。

**查询参数**

| 参数               | 类型     | 必填 | 默认值 | 约束            | 说明                                                         |
| ------------------ | -------- | ---- | ------ | --------------- | ------------------------------------------------------------ |
| `keyword`          | `string` | 否   | -      | -               | 关键字搜索，同时匹配模型名称和描述                           |
| `modelType`        | `int`    | 否   | `0`    | `0`、`1`        | 模型类型过滤。`0` 表示查询 `model_type` 为 `0` 和 `1` 的文本对话模型；`1` 表示仅查询 `model_type = 1` |
| `supportInference` | `bool`   | 否   | `true` | `true`、`false` | 是否支持推理。`true` 表示仅返回存在 `serviceId` 的模型，`false` 表示仅返回 `serviceId` 为空的模型 |
| `page`             | `int`    | 否   | `1`    | `≥ 1`           | 页码，从 1 开始                                              |
| `page_size`        | `int`    | 否   | `20`   | `1 ~ 100`       | 每页返回记录数                                               |

**查询规则说明**

- 当 `modelType` 未传或为 `0` 时，返回 `model_type IN (0, 1)` 的模型
- 当 `modelType=1` 时，仅返回 `model_type = 1` 的模型
- 当 `modelType` 为其他值时，返回 `10001 invalid parameter`
- 仅返回 `atp_file_storage.is_public = 1` 的公开模型
- 当 `supportInference` 未传时，默认按 `true` 处理
- 当 `supportInference=true` 时，仅返回 `serviceId` 非空的模型
- 当 `supportInference=false` 时，仅返回 `serviceId` 为空的模型

**响应**

成功时返回 HTTP 200：

```json
{
  "code": 0,
  "message": "success",
  "data": [
    {
      "name": "Spark Lite",
      "description": "星火轻量文本对话模型",
      "model": "spark-lite",
      "id": "2402509314320990"
    },
    {
      "name": "DeepSeek Chat",
      "description": "通用文本对话模型",
      "model": "deepseek-chat",
      "id": "2402509314320991"
    }
  ],
  "total": 2
}
```

**`data[*]` — ModelResponse 模型响应**

| 字段          | 类型     | 说明                             |
| ------------- | -------- | -------------------------------- |
| `name`        | `string` | 模型名称                         |
| `description` | `string` | 模型描述                         |
| `model`     | `string` | 映射自 `serviceId` |
| `id`     | `string` | 模型服务标识 |


**错误响应示例**

当 `modelType` 不在支持范围内时：

```json
{
  "code": 10001,
  "message": "invalid modelType: 2"
}
```
